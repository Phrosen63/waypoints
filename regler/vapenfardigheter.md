# Vapenfärdigheter

Vapenfärdigheter representerar hur skicklig en karaktär är med olika typer av vapen. Om en karaktär använder ett vapen utan rätt vapenfärdighet får de en **-3-modifikation** på **attack**, **parering**, **blockering** och **skada**.

## Förkortningar

| Förkortning | Betydelse |
|---|---|
| VF | Vapenfärdighet |
| 1h | Enhandsvapen |
| 2h | Tvåhandsvapen |


Exempel: En spelare kan skriva **VF: 1h yxor** på karaktärsbladet för att markera färdighet med enhandsyxor.

## Vapenfärdigheter

Följande vapenfärdigheter finns:

* **1h långa klingor**: \
Vapen med klingor längre än 45 cm, lämpade för både hugg och stick. De är mångsidiga och erbjuder bra räckvidd, populära bland soldater och krigare. \
Exempel: Långsvärd, bredsvärd, sabel, falchion, rapir, katana
* **1h korta klingor: \
**Vapen med klingor under 45 cm. Lättare att dölja och använda i trånga utrymmen. Passar både som huvudvapen och reservvapen. \
Exempel: Dolk, kniv, stilett, kortsvärd, machete, kukri
* **1h yxor: \
**Kompakta och effektiva vapen för huggattacker. Snabba att svinga och ofta användbara i närstrid. \
Exempel: Stridsyxa, handyxa
* **1h påkvapen: \
**Enhandsvapen som orsakar trubbigt trauma genom slag. De är effektiva mot rustningar och lämpar sig väl för att krossa ben eller förstöra utrustning. \
Exempel: Stridsklubba, morgonstjärna, kort stridshammare
* **Kastvapen: \
**Vapen designade för att träffa mål på avstånd genom kast. \
Exempel: Kastkniv, stridsspjut, chakram, kaststjärna
* **2h långa klingor: \
**Tvåhandssvärd med långa klingor som erbjuder stor räckvidd och kraftfulla hugg. Kräver styrka men är effektiva mot rustning. \
Exempel: Zweihänder, claymore, flamberge
* **2h yxor: \
**Tunga vapen för tvåhandsgrepp, utmärkta för att orsaka stor skada vid hugg. \
Exempel: Dubbeleggad yxa, bardiche
* **2h påkvapen: \
**Tunga tvåhandsvapen som levererar kraftfulla slag och kan slå igenom även de mest robusta rustningar. Dessa vapen kräver styrka och är särskilt användbara mot fiender med tungt skydd. \
Exempel: Släggliknande stridshammare, stor morgonstjärna, dubbelsidig stridsklubba
* **2h stångvapen:** \
Långa vapen med brett användningsområde, både för att hålla avstånd och leverera kraftfulla attacker. \
Exempel: Spjut, glaive, poleaxe
* **Pilbågar: \
**Avståndsvapen som kräver precision och skicklighet. Effektiva mot fiender på avstånd. \
Exempel: Långbåge, kortbåge, recurvebåge
* **Armborst: \
**Kraftfulla projektilvapen som är lättare att använda än pilbågar men långsammare att ladda. \
Exempel: Lätt armborst, tungt armborst, repetearmborst

## Sköldar som vapenfärdighet

Sköldar kräver också vapenfärdighet och kan användas både defensivt och offensivt. De delas in i följande kategorier:

* **Små sköldar \
**Lätta och snabba att använda, ofta för parering. \
Exempel: Bucklare \
IM: +-0
* **Medelstora sköldar: \
**Erbjuder en balans mellan skydd och rörlighet. \
Exempel: Rundsköld, vikingasköld, draksköld (kite shield), stridssköld \
IM: +1
* **Stora sköldar: \
**Ger högt skydd men är tyngre att bära och påverkar rörligheten. \
Exempel: hoplitersköld, gotisk sköld \
IM: +2
* **Tornsköldar: \
**Massiva sköldar som täcker stora delar av kroppen. Perfekta för försvar men kräver mycket styrka och uthållighet. \
IM: +4

Notera: IM för sköldar ovan är endast rekommendationer. Magiska sköldar, som till exempel en förtrollad rundsköld, kan ha lägre IM än vad listan antyder.
